# Academic Projects - Computer Science

Welcome to my portfolio of academic projects. This repository showcases my work during my Computer Science studies at the Technion. 

My primary focus in these projects has been on **system-level C++ development**, implementation of **advanced data structures from scratch**, and ensuring optimal **algorithmic complexity** under strict constraints.

## Core Competencies
* **Advanced Data Structures:** Implementation of AVL Trees, HashMaps, and Disjoint Set Union (DSU) with Path Compression.
* **C++ Proficiency:** Robust code design, memory management (RAII, `std::shared_ptr`, `std::unique_ptr`), and adherence to modern C++ standards.
* **Complexity Analysis:** Theoretical and practical optimization of time and space complexity for high-performance systems.
* **Software Engineering:** Object-Oriented design, modular architecture, and unit-testing awareness.
